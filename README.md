## CNN quantification for image classification

#### create venv
~~~
$ virtualenv --system-site-packages -p python3 ./venv
~~~
#### activate venv
~~~
$ source ./venv/bin/activate

(venv) $ pip install --upgrade pip

(venv) $ pip install -r requirements.txt

(venv) $ pip list 
~~~
#### deactivate and exit
~~~
(venv) $ deactivate
~~~
